## texplates
Simple and functional LaTeX document templates 
### Dependencies
* pdflatex (usually part of TeXLive)
* (optional) ImageMagick for converting to png/jpg
* (optional) make
### Example usage
```
make coverletter.pdf
make coverletter.png
```
